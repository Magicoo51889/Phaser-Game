# Phaser - Typescript - Webpack
This project uses these three packages to be able use phaser with modern programming practices, which allows for the game to be better developed and easier to debug and code with. 

### Coding with Phaser
Phaser's documentation is quite outdated in places so can be difficult to find solutions to some things, and can be a headache to fix. Thankfully typescript is able to give more detailed syntax highlighting and errors which helps to combat this, and webpack allows me to develop the game locally, and quickly be able to change and adjust code to test out different solutions, and test how things work.
