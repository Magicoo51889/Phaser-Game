# Phaser - Typescript - Webpack
This project uses these three packages to be able use phaser with modern programming practices, which allows for the game to be better developed and easier to debug and code with. 

### Coding with Phaser
Phaser's documentation is quite outdated in places so can be difficult to find solutions to some things, and can be a headache to fix. Thankfully typescript is able to give more detailed syntax highlighting and errors which helps to combat this, and webpack allows me to develop the game locally, and quickly be able to change and adjust code to test out different solutions, and test how things work. 

### Using Webpack
Webpack is a brilliant package that allows me to develop the game locally, and quickly be able to change and adjust code to test out different solutions, and test how things work. It also allows me to use the latest version of Phaser, which is a great for developing the game whilst keeping it up to date.

### Using TypeScript
Phaser doesn't use typescript as standard, but they allow you to use it with a change in the configuration of Phaser. This allows me to get the benefits of typescript without having to change masses of code. I also use typescript to make the code more readable and easier to understand as it has a lot of syntax highlighting and error checking which helps to find and fix bugs before they even happen.
